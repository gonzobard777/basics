# Unix комманды

| Команда                                                                | Описание                                                                                                                                                                                                                                |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| &nbsp;                                                                 | &nbsp;                                                                                                                                                                                                                                  |
| `find / -name \*.mbtiles 2>&1 \| grep -v 'Permission denied'`          | поиск файлов с игнором Permission denied                                                                                                                                                                                                |
| &nbsp;                                                                 | &nbsp;                                                                                                                                                                                                                                  |
| wget -c --tries=0 --retry-connrefused --waitretry=10 "ССЫЛКА" -P ПАПКА | `-c` — докачка с места обрыва, `--tries=0` — бесконечные повторы, `--waitretry=10` — пауза 10 с между попытками. Если совсем прервётся (закрыли терминал и т.п.) — просто запустите ту же команду снова, она продолжит с того же места. |
